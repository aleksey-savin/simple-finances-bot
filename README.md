# Simple Finances Bot

## Deployment

### Prerequisites
- Docker
- Docker Compose

### Setup
1. Clone the repository:
```bash
git clone https://github.com/aleksey-savin/simple-finances-bot.git
cd simple-finances-bot
```
2. Modify the .env.example file and save it as .env
3. Run deploy.sh script:
```bash
chmod +x deploy.sh
./deploy.sh
```
